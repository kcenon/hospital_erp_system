## 5. Related Regulations Summary

### 5.1 Personal Information Protection Act

```
┌─────────────────────────────────────────────────────────────────┐
│              Personal Information Protection Act Key Articles    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Article 23 (Restrictions on Processing Sensitive Information)  │
│  - Separate consent required for sensitive info (health,        │
│    genetics, criminal records)                                   │
│  - Medical information is classified as sensitive                │
│                                                                  │
│  Article 24 (Restrictions on Processing Unique Identifiers)     │
│  - SSN collection only permitted when allowed by law            │
│  - Encrypted storage required                                    │
│                                                                  │
│  Article 29 (Safety Measures)                                    │
│  - Technical, administrative, physical protection measures      │
│  - Access rights management, access record retention            │
│                                                                  │
│  Article 34 (Notification of Personal Information Leakage)      │
│  - Notify data subjects immediately upon breach                 │
│  - Report to Privacy Protection Commission if 1,000+ affected   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 Medical Service Act

```
┌─────────────────────────────────────────────────────────────────┐
│                    Medical Service Act Key Articles              │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Article 21 (Record Inspection)                                  │
│  - Medical personnel/institutions must restrict record access   │
│    to persons other than patients                                │
│  - Exceptions only for patient consent or legal requirements    │
│                                                                  │
│  Article 22 (Medical Records)                                    │
│  - Obligation to create and retain medical records              │
│  - Digital signature required for EMR systems                   │
│                                                                  │
│  Article 23 (Electronic Medical Records)                         │
│  - Must ensure authenticity, integrity, and security            │
│  - Comply with standards set by Ministry of Health              │
│                                                                  │
│  Retention Periods                                               │
│  - Medical records: 10 years                                    │
│  - Surgical records: 10 years                                   │
│  - Nursing records: 5 years                                     │
│  - Patient registry: 5 years                                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```
