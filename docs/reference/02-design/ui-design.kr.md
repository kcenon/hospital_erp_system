## 1. 디자인 원칙

### 1.1 핵심 원칙

| 원칙 | 설명 | 적용 |
|------|------|------|
| **명확성** | 정보의 명확한 전달 | 직관적 레이아웃, 명확한 레이블 |
| **효율성** | 최소 클릭으로 업무 완료 | 단축키, 빠른 입력 |
| **일관성** | 동일한 패턴과 컴포넌트 | 디자인 시스템 준수 |
| **접근성** | 모든 사용자 접근 가능 | WCAG 2.1 AA 준수 |
| **반응성** | 다양한 기기 지원 | PC, 태블릿, 모바일 최적화 |

### 3.2 반응형 Breakpoints

| Breakpoint | 너비 | 레이아웃 | 대상 기기 |
|------------|------|----------|----------|
| Mobile | < 640px | 단일 컬럼, 하단 탭 | 스마트폰 |
| Tablet | 640px - 1023px | 2컬럼, 축소 사이드바 | 태블릿 (회진용) |
| Desktop | 1024px - 1279px | 풀 레이아웃 | 노트북 |
| Wide | ≥ 1280px | 확장 레이아웃 | 대형 모니터 |
